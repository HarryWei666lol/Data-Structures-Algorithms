Substitution


Using substitution, prove that the solution to T(n)=T(n-1)+10 is T(n)=10n

Base Case: 
 T(1) = 10

Inductive Step:
 Assume the proposition holds true for 1<= n <=p.
 T(n-1) = 10(n-1)
 T(n) = T(n-1)+10
 T(n) = 10(n-1)+10
 T(n) = 10n

QED.

Using substitution, show that T(n)=T(n-1)+10 is O(n-squared)

Base Case:
 T(1) = 10
 n = 1, c = 10, n^2=1
 T(1)<=cn^2<=10=O(n^2)

Inductive Step:
 Assume the proposition holds true for 1<= n <=p.
 T(n-1) is O(n^2).
 T(n)=T(n-1)+10
 T(n)=O(n^2)+10
 T(n)=O(n^2)+c, c=10
 Ignore c as c is a constant.
 T(n)=O(n^2)

QED.
